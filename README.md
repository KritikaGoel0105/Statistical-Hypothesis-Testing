## 📌 Statistical Tests Included

### ✅ T-Test
Used to compare the means of two groups.

### ✅ Z-Test
Used when population variance is known or sample size is large.

### ✅ Chi-Square Test
Used for categorical data analysis and independence testing.

### ✅ ANOVA
Used to compare means across multiple groups.

### ✅ One-Tailed & Two-Tailed Tests
Used for directional and non-directional hypothesis testing.

### ✅ Outlier Detection
Detection of anomalies using statistical methods.
